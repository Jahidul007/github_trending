# github_trending

### How to run


###### Prerequisite
- Flutter 3.7.3 (Stable)
- Dart 2.19.2

###### Build
```
$ flutter build apk --release
```

###### Run
```
$ flutter run 
```

### Github api documentations
```
https://docs.github.com/en/rest
```
